## 📌 Overview
A classic Battleship game. Place your ships and sink your enemies ships before they sink yours.

## 🛠️ Tech Stack
- Frontend: Javascript

## 🚀 Features
- Place your ships on the grid
- Attack enemy ships by selecting fields on the opponents grid
